O que é OLAP ?

Os estatisticos e cientistas de dados frequentemente precisam agrupar e agregar dados. Essas operações em bancos de dados relacionais consomem muitos recursos. Com o OLAP(On-Line Analytical Processing), os dados podem ser pré-calculados e agregados, tornando a análise mais rápida.

OLAP é uma categoria de software que permite aos usuários analisar informações de vários sistemas de banco de dados ao mesmo tempo. É uma tecnologia que permite aos analistas extrair e visualizar dados de diferentes pontos de vista. OLAP significa Processamento analítico online

Cubo OLAP:

No centro do OLAP, o conceito é um cubo OLAP ou cubo de dados. O cubo OLAP é uma estrutura de dados otimizada para análise de dados muito rápida e eficiente. O cubo OLAP consiste em instancias numéricas chamadas medidas que são categorizadas pelas dimensões. O cubo OLAP também é chamado de hipercubo. Normalmente, as operações e análises de dados são realizadas usando uma planilha simples, onde os valores dos dados são organizados em formato de linhas e colunas. Isso é ideal para dados bidimensionais. No entanto, o OLAP contém dados multidimensionais, com dados geralmente obtidos de fontes diferentes e não relacionadas. Usar uma planilha não é uma opção ideal. O cubo pode armazenar e analisar dados multidimensionais de maneira lógica e ordenada.

Como funciona? Um data warehouse extrairia informações de várias fontes e formatos de dados, como arquivos de texto, planilha do excel, arquivos multimídia etc. Os dados extraídos são limpos e transformados. Os dados são carregados em um servidor OLAP (ou cubo OLAP) em que as informações são pré-calculadas com antecedência para análise posterior. Vantagens do OLAP

O OLAP é uma plataforma para todos os tipos de negócios, incluindo planejamento, orçamento, geração de relatórios e análise entre outros. Informações e cálculos são consistentes em um cubo OLAP. Este é um beneficio importante.. Pesquisa facilmente no banco de dados OLAP termos gerais ou específicos. O OLAP fornece os blocos de construção para ferramentas de modelagem de negócios, ferramentas de mineração de dados, ferramentas de relatório de desempenho. É bom para analisar séries temporais. Encontrar alguns clusters e outliers é fácil com o OLAP. É um poderoso sistema de processo analítico de visualização on-line que fornece tempos de resposta mais rápidos

Desvantagens do OLAP

O OLAP requer a organização dos dados em um esquema em estrela ou floco de neve. Esses esquemas são complicados de implementar e administrar Você não pode ter um grande número de dimensões em um único cubo OLAP Os dados transacionais não podem ser acessados ​​com o sistema OLAP.
